This folder contains scripts which compare cluster maps between HCHO and the other variables. Clusters are compared individually, with the southwest coast, eastern coast, and northern coast being chosen for comparisons. Scripts included are:
* HCHO and EF closeup Clustering Difference Map
* HCHO and LAI closeup Clustering Difference Map
* HCHO and PFT closeup Clustering Difference Map
* HCHO and SM closeup Clustering Difference Map
